# Skill — Cálculos Trabalhistas Completos

Esta skill foi criada para realizar **cálculos trabalhistas completos**, permitindo ao usuário escolher entre:

- **Cálculo Automático**  
- **Cálculo Assistido**

A skill solicita dados essenciais, faz perguntas obrigatórias e entrega um cálculo completo com explicações jurídicas.

---

## Funcionalidades

- Cálculo de:
  - Verbas rescisórias
  - Horas extras
  - Adicional noturno
  - Insalubridade
  - Comissões
  - Férias + 1/3
  - 13º salário
  - FGTS + Multa
  - Reflexos e integrações

- Fluxo inteligente com perguntas obrigatórias:
  - Recebia horas extras? Quanto  
  - Trabalhava entre 22h e 5h  
  - Recebia comissão  
  - Recebia algum valor por fora  
  - Fez algum acordo por fora
